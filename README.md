# Susobhan Portfolio Website

This is a multi-page portfolio website ready for **GitHub Pages** deployment.

## Files included
- `index.html`
- `about.html`
- `projects.html`
- `contact.html`
- `style.css`
- `script.js`

## Deploy on GitHub Pages
1. Create a new GitHub repository (for example: `portfolio`).
2. Upload all files from this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/root`
5. Save and wait 1–5 minutes.
6. Your website will be live at:
   `https://your-username.github.io/portfolio/`
