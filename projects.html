Projects & Research Work

Selected machine learning and data-driven projects based on your CV.

Hate Speech Detection Using NLP and LSTM

Developed an automated system to classify tweets as hate speech or non-hate speech using NLP and a multi-layered LSTM model. Included embedding, dropout, and dense layers to reduce overfitting. Achieved 98.38% training accuracy and 97.08% validation accuracy. Deployed with Gradio.

Bird Sound Classification Using CNN

Built a CNN-based bird voice classification model for 30 bird species. Extracted MFCC audio features to capture temporal and frequency patterns. Achieved 95.67% validation accuracy.

Salary Prediction Using Linear Regression

Created salary prediction models using simple and multiple linear regression. Performed missing value handling, encoding, outlier detection, and feature selection on 6,700+ records. Achieved R² score of 0.695 for multiple linear regression.

Internship Highlight — Feynn Labs

Worked as a Machine Learning Intern (July 2024 – Sept 2024). Conducted Indian EV market segmentation using PCA, K-Means clustering, and Agglomerative Hierarchical Clustering to identify actionable customer groups and support targeted market entry strategies.
